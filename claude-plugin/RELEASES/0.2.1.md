# secondbrain-db plugin v0.2.1

**Patch release.** No behavior changes; pairs with `secondbrain-db` v1.1.1 (CI + tooling improvements only).

## What's in this release

No code changes in the plugin itself. The bump aligns the plugin's published version with the corresponding sbdb-db CLI release that landed alongside it.

## Coordinated CLI release

The companion `secondbrain-db` v1.1.1 ships:

- Aligned CI Go version with `go.mod` (no more flaky toolchain auto-downloads under load)
- Expanded GitHub Actions matrix: format, govulncheck (advisory), stress concurrency, e2e suite against the built binary
- README polish — marketing tagline, audience list, ASCII component / write-path / archival / worker diagrams

No changes in plugin behavior; the `guard-docs.py` and `guard-events.py` hooks, integrity checks, skills, agents, and commands are byte-identical to v0.2.0.

## Upgrade

```
/plugin update secondbrain-db
```

Or, for the CLI:

```
go install github.com/sergio-bershadsky/secondbrain-db@latest
```
